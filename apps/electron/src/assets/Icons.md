# Electron App Icons

Place your app icons in this directory:

- `icon.png` - 512x512 PNG icon for Linux
- `icon.ico` - Windows icon (include 16x16, 32x32, 48x48, 256x256)
- `icon.icns` - macOS icon

You can use tools like:
- [electron-icon-builder](https://github.com/safu9/electron-icon-builder)
- [electron-forge-maker-appimage](https://github.com/electron-userland/electron-forge-maker-appimage)

To generate icons from a single source image.
