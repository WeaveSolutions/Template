# Tauri Icons

This directory should contain the following icon files for your application:

- `32x32.png` - 32x32 pixels
- `128x128.png` - 128x128 pixels  
- `128x128@2x.png` - 256x256 pixels (for Retina displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

You can generate these icons from a single high-resolution source image using:
```bash
pnpm tauri icon path/to/source-icon.png
```

The source icon should be at least 1024x1024 pixels for best results.
